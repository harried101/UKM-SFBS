# UKM Sports Facilities Booking System

## Setup Instructions

1. Install XAMPP and start Apache + MySQL.
2. Create database `ukm_sports_booking` in phpMyAdmin.
3. Import `database/ukm_sports_booking.sql`.
4. Place this project in `C:\xampp\htdocs\ukm-sfbs`.
5. Open `http://localhost/ukm-sfbs/public/index.php` in browser.
